# Inception

This project aims to teach the fundamentals of Docker and how to use it with different services, such as MariaDB, WordPress, and Nginx.

Given the scope of the project and the number of key concepts involved, a detailed explanation is provided on the following Notion page:

## Installation
- Run make
- Create the .env file
- Enjoy the website

```bash
  make
  touch .env [Add the data inside the file]
```
